## Projeto: feedback-widget

# Camada: Front-end

# Tecnologia: React

# Ferramenta:

    Node.Js, Estrutura Vite(React + Typescript)

# Biblioteca:

    tailwindcss
    phosphor icon
    headless-ui

# Comandos:

    `yarn create vite`
    `yarn dev`
    `yarn add -D tailwindcss postcss autoprefixer`
    `yarn tailwindcss init -p`
    `yarn add phosphor-react`
    `yarn add @headlessui/react`
